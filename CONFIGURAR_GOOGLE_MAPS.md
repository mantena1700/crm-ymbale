## Como Configurar

1. Abra o arquivo `.env.local` na raiz do projeto
2. Adicione a seguinte linha:
```
NEXT_PUBLIC_GOOGLE_MAPS_API_KEY=Ucc4RbcoUyECWqqXkwCrJNlSZhQ=
```

3. Reinicie o servidor Next.js para que as mudanças tenham efeito
